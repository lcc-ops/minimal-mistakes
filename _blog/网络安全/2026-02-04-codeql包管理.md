# 包管理



- 初始化

  ```
  codeql pack init fastjson
  ```

  在fastjson目录下会生成一个qlpack.yml

- 添加依赖

  ```
  codeql pack add codeql/java-all
  ```

  这会添加相关的依赖并下载

- 下载依赖

  ```
  codeql pack install
  ```

  会根据当前目录下的qlpack.yml下载需要的依赖，依赖会下载到用户根目录下的`.codeql\packages`
